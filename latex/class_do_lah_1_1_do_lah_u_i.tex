\subsection{Detailed Description}
Setup the main window for the \hyperlink{namespace_do_lah}{Do\+Lah} application. 

The main window is made up of 3 main components\+: the menu, \hyperlink{class_do_lah_1_1_display_area}{Display\+Area}, and \hyperlink{class_do_lah_1_1_input_box}{Input\+Box}.
